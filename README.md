# VIT Door-to-Door Delivery Service

## Project Overview
The **VIT Door-to-Door Delivery Service** is a Python-based tool designed to streamline the delivery of items within the VIT Boys Hostel. It connects students with a delivery service for sports equipment, study materials, beverages, and snacks. The system calculates the total cost based on item price and quantity, adds a fixed delivery fee, and takes the Block and Room Number for efficient delivery.

## Features
* **Service Menu:** Displays available categories like sports items, books/notes, and food/beverages.
* **Order Processing:** Takes item details, quantity, and unit price from the user.
* **Cost Calculation:**

## Installation & Execution
Follow these steps to set up and run the project on your local machine:

1.  **Prerequisites:** Ensure you have Python 3 installed.
2.  **Download the Project:** Save the `main.py` file to your local folder.
3.  **Run the Application:** Open your terminal or command prompt, navigate to the folder, and run:
    ```bash
    python main.py
    ```

## Testing Instructions
To verify the system works correctly, try the following test case:

* **Input Item:** `Chips`
